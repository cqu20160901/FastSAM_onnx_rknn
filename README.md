# FastSAM_onnx_rknn
FastSAM  部署版本，便于移植不同平，部署简单、运行速度快。


FastSAM_onnx：onnx模型、测试图像、测试结果、测试demo脚本

FastSAM_rknn：rknn模型、测试（量化）图像、测试结果、onnx2rknn转换测试脚本
